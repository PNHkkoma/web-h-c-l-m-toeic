Dự án sử dụng Nodejs để code back-end, Js và HTML + CSS để code giao diện

Web có các chức năng như:
 - Đăng nhập, đăng xuất
 - Thêm sửa xóa bài học
 - Thêm sửa xóa người dùng
 - Làm bài học

 
cách chạy code:
- Bước 1:
```
git clone 
```
- Bước 2:
```
cd web_ban_mo_hinh
```
-  Bước 3:
```
yarn i
```
or
```
npm install
```
-  Bước 4:
```
npm start
```
